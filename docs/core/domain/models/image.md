# Модуль image

Модуль содержит доменную модель загруженного изображения.

## Класс UploadedImage

**Доменная модель загруженного изображения.**

#### Аргументы
| Аргумент | Тип | Описание |
|----------|-----|----------|
| `_id` | `ObjectId` | Уникальный идентификатор изображения |
| `user_id` | `ObjectId` | Уникальный идентификатор пользователя |
| `url` | `str` | Путь к изображению |
| `uploaded_at` | `int` | Дата загрузки изображения |

```python
@dataclass
class UploadedImage:
    """Доменная модель загруженного изображения.

    Attributes:
        _id (ObjectId): Уникальный идентификатор изображения
        user_id (ObjectId): Уникальный идентификатор пользователя
        url (str): Путь к изображению
        uploaded_at (int): Дата загрузки изображения
    """

    _id: ObjectId
    _user_id: ObjectId
    url: str
    uploaded_at: int
```

## def id:
#### Возвращает строковое представление ID.

#### Возвращает
| Тип | Описание |
|-----|----------|
| `str` | Уникальный идентификатор картинки |

```python
    @property
    def id(self) -> str:
        """Возвращает строковое представление ID.

        Returns:
            str: Уникальный идентификатор картинки
        """
        return str(self._id)
```
