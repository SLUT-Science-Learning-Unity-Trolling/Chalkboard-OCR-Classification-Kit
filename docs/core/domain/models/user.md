## Класс User

**Доменная модель пользователя.**

**Args:**
- `_id`: Уникальный идентификатор пользователя
- `username`: Имя пользователя
- `email`: Email пользователя

---
## id:
#### Возвращает строковое представление ID.

#### Возвращает
| Тип | Описание |
|-----|----------|
| `str` | Уникальный идентификатор пользователя |

```python
    @property
    def id(self) -> str:
        """Возвращает строковое представление ID.

        Returns:
            str: Уникальный идентификатор пользователя
        """
        return str(self._id)
```
